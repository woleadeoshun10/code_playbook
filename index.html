How the Internet works?

When we browse or "surf" the Internet from our computer (the client), this is what happens behind the scenes:

  • The browser sends a request for a webpage (e.g., www.example.com).
  • This request goes to our Internet Service Provider (ISP).
  • The ISP forwards the request to a Domain Name System (DNS) server.
  • The DNS acts like a phone book — it translates the human-friendly domain name (like example.com) into the exact IP address of the web server.
  • Once the IP address is found, the request is sent to the web server hosting the site.
  • The server responds by sending the requested webpage back to the client’s browser.
  • The browser then renders the page for the user to see.

We can look up an ip address by going to Nslookup.io.
